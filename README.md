# HelloWorld Example

```
pip3 install aurorastack-core
pip3 install aurorastack-api
```

# Download Project

```
git clone https://github.com/aurorastack/helloworld.git
```

# Run Server

```
cd helloworld

# Run gRPC Server
aurorastack run grpc-server myproject -m src/

# Run rest Server
aurorastack run rest-server myproject -m src/
```
